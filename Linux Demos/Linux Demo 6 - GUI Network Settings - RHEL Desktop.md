# Linux Demo 6

## Review IPv6 Configuration Options with a Linux Desktop Instance 

1. Connect and login to the RHEL desktop.
2. Open the **Settings** application and select **Network**.
3. Click on the gear icon next to enable/disable slider for the primary interface and observe the IPv6 information (if any) on the **Details** tab.
